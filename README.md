# EGG

EGG est une institution éthique multi-environnement d'accompagnement par l'intelligence artificielle, en cours de construction. Ce dépôt rassemble les documents fondateurs et publics du projet — il n'héberge pas le code de la plateforme, qui reste pour l'instant dans un dépôt privé.

## La démarche

EGG poursuit une thèse simple : *l'accompagnement assisté par l'IA peut être pratiqué avec rigueur si le cadre dans lequel il s'inscrit est lisible, opposable, et préfère retirer plutôt que séduire*. Cette thèse se traduit dans une plateforme dont l'éthique est inscrite par construction — pas par communication — et dont chaque engagement est vérifiable au regard d'un texte fondateur public, la **Constitution EGG**.

## La Constitution

La Constitution EGG est le document fondateur du projet. Elle fixe ce qui n'est pas négociable : la posture, les invariants techniques, le modèle économique, l'isolement structurel des contextes, la voix moyenne, les protocoles pour les situations limites. Toute évolution de la plateforme s'inscrit dans son cadre, ou elle révise la Constitution selon une procédure publique et préavisée.

Lecture de la Constitution en vigueur (v3.1, mai 2026) : [egg-constitution-v3_1.md](./egg-constitution-v3_1.md).

L'historique complet des versions est tracé dans le préambule du document, et les modifications sont versionnées via Git — ce dépôt est l'archive opposable.

## Engagement de continuité

Si la société qui exploite EGG cesse son activité, le code source de la plateforme — à l'exception des agents conçus par les eggsperts, qui restent leur propriété — sera publié dans ce dépôt en open source. Cela permet une ré-implémentation par un tiers, à laquelle les eggsperts pourraient revenir publier leurs agents si la nouvelle implémentation respecte la Constitution.

Ce mécanisme garantit que l'arrêt d'EGG ne signifie pas la disparition de l'usage : la plateforme survit comme code disponible, et les utilisateurs qui auraient construit un parcours peuvent le récupérer (via les fonctions d'export prévues par la Constitution) avant que les serveurs ne soient éteints.

## Signaler un manquement

Toute violation présumée d'un engagement constitutionnel peut être signalée à `maxime@simplex.coach`. Les manquements concernant un engagement structurel (anonymat, économie, données, souveraineté) sont relayés en *issue publique* sur ce dépôt — leur résolution est suivie publiquement.

Les bugs techniques relèvent d'un autre canal qui sera précisé en temps voulu.

## Cadre déontologique externe

EGG souscrit au [Coaching Trust Framework](https://coaching-trust-framework.org/), avec sa déclinaison ICF pour les eggspériences de coaching. Le CTF est un cadre ouvert développé en parallèle d'EGG, applicable à toute plateforme d'accompagnement par IA — il est publié indépendamment.
